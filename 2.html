块元素排序

02/03/2018

块元素的排序用z-index的属性，当没有添加这个CSS 属性时，块元素会按如下顺序堆叠（从底到顶）：
1.根元素的背景和边界；
2.没有任何定位的块元素（没有position或者position: static） 按HTML中的出现顺序堆叠；
3.有定位的块元素，同样按HTML中的出现顺序堆叠。

如何用CSS来给块元素排序叠加呢？

1.如果想要自己控制块元素的堆叠的顺序，可以给这些块元素添加z-index的CSS属性， 其属性值自己排序，那么就会按你写的顺序来堆叠了。
2.如果想要有元素块重复叠加的效果，那么在给这些需要叠上去的元素添加top/bottom/left/right 的属性，其属性值根据你想要叠加上去的数量值来写。

用HTML如何做用户填写栏？

01/03/2018

<label for="duanxing">message</label>
<textarea name="duanxing"></textarea>

注：for和name的属性值是一样的
<textarea>这个中间切记不能填写任何东西,这是留给用户写的。</textarea>

如何用HTML做下拉选项框？

01/03/2018

<label for="xingbie">sexe</label>
<select name="xingbie">
    <option value="M">Homme</option>
    <option value="F">Femme</option>
</select>

注：
1，for和name的属性值必须一样；
2，option可以写很多条，但是必须要添加value的属性值。

如何用HTML做固定不变的表格？

01/03/2018

用<table>的元素来做，线条框框必须用CSS来做。

<table>
<tr>
    <th></th><!-- th为了最上面或者最边上的标签(表头单元格) -->
    <th>年龄</th>
    <th>身高</th>
    <th>体重</th>
</tr>
<tr>
    <td>A同学</td><!-- 普通单元格 -->
    <td>15岁</td>
    <td>170cm</td>
    <td>50kg</td>
</tr>
<tr>
    <td>B同学</td>
    <td>13岁</td>
    <td>156cm</td>
    <td>38kg</td>
</tr>
<tr>
    <td>C同学</td>
    <td>16岁</td>
    <td>178cm</td>
    <td>55kg</td>
</tr>
</table>
                
table {
    border: 2px solid plum;
    /* 表格边框分开还是合并的属性 */
    border-collapse: collapse;
    width: 100%;
}

table td,
table th {
    text-align: center;
    border: 2px solid plum;
}
