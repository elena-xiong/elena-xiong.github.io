CSS伪类 Pseudo-class

04/03/2018

CSS伪类是一个以冒号（：）作为前缀关键字的，是让指定的元素在特定的状态下 呈现的样子。
常用的有：

/* 鼠标放在上面发生改变 */
:hover {...} 

/* 这两个伪元素，是在同一个父元素内有几个并列的子元素时，可以用这个
来准确改变子元素的值（如：位置，颜色，大小等等）*/
:nth-of-type(odd) {...} 奇数
:nth-of-type(even) {...} 偶数

/* 是按鼠标时发生改变，多数和a，button一起用 */
:active {...}

/* 是链接地址已经看过后发生改变，一般是颜色的改变，只能
和a一起使用
:visited {...}

CSS自定义选择器

04/03/2018

在HTML里面用data-自定义名称来给元素自定义一个属性，后面 也可以添加"=..."来定义一个属性值，然后为了改变这个元素 在CSS里用： 

/* 该选择器包含了data-自定义名称属性的所有元素，不管其属性值是什么 */                        
[data-自定义名称] {...} 

/* 该选择器是仅选择了data-自定义名称属性中属性值为"..."的元素 */
[data-自定义名称="..."] {...}

/* 该选择器是仅选择了data-自定义名称属性中属性值为"..."的元素，
以空格间隔了多个属性值 */
[data-自定义名称~="..."] {...}

CSS选择器 Selectors

04/03/2018

A , B {...} 是A与B同时选择；
<p>Lorem ipsum dolor sit amet</p>
<strong>consectetur adipiscing elit</strong>
                    
p , strong {
    color: yellow;
} /* p和strong一起改变 */
A B {...} 是选择A里面的B；
<div>
    <p>我会变</p>
    <article>
        <p>我也会变</p>
    </article>
</div>
                    
div p {
    color: yellow;
} /* 改变了p */
A > B {...} 是选择了B（B是A的子元素）；
<div>
    <p>我会变</p>
    <article>
        <p>我不会变</p>
    </article>
</div>
                    
div > p {
    color: yellow;
} /* 改变了子元素p */
A + B {...} 是选择了B（B是A的下一个兄弟元素）；
<div> consectetur adipiscing elit</div>
<p>Lorem ipsum dolor sit amet</p>
                    
div + p {
    color: yellow;
} /* 改变了p */
A ~ B {...} 是选择了B（B是A的下面的兄弟元素们）； 
<div>
    <p>Lorem ipsum dolor sit amet</p>
    <strong>consectetur adipiscing elit</strong>
</div>
                    
div ~ p, div ~ strong {
    color: yellow;
} /* 改变了p和strong */

box-sizing

03/03/2018

这个CSS属性用于更改计算元素宽度和高度的默认CSS 盒子模型。
box-sizing: content-box;是默认，一般 不需要写；
box-sizing: border-box;是告诉浏览器 去理解你设置的边框和内边距的值是包含在width内的。
注：
为了方便，大家一般会把 * { box-sizing: border-box; }写在CSS最前面。

如何用CSS控制盒子的背景

02/03/2018

给盒子添加以下这些CSS属性，其属性值自己改变

div {
    /* 背景色 */
    background-color: yellow;

    /* 渐变效果 */ 
    background-image: linear-gradient (to bottom, yellow, #dddd00, 50%, orange);

    /* 放正常图片 */
    background-image: url(......);

    /* 图片小于盒子时，默认会重复，其属性值no-repeat选择不重复 */
    background-repeat: no-repeat;

    /* 背景的位置，控制上下左右,跟图片一起使用*/
    background-position: 99% center;
}
