如何让FOOTER一直保持在网页最下面？

08/03/2018

1.给html一个css属性值为： 
html {
    min-height: 100%; /* 最小高度是浏览器的最大高度 */
    position: relative; /* 是为了可以和footer用absolute */
}
2.给footer一个css属性值为： 
footer {
    heright: 100px;
    position: absolute;
    bottom: 0;
    width: 100%;
}
3.给body一个css属性值为： 
body {
    /* 这个数值必须和footer的height值一样，这样footer就不会挡住正文内容 */
    margin-bottom: 100px;
}

CSS的position属性

06/03/2018

static是默认值，任意的元素不会被特定位 
{
    position: static;
}
fixed属性值会相对于视窗来定位，这意味着页面滚动，但是这个视窗不会动，停在相同的位置。 
{
    position: fixed;
    bottom: 0;
    right: 0;
    width: 200px;
    background-color: white;
}
relative属性值表现的和static一样，除非额外添加属性，后面四个就是额添加的控制大小，位置和颜色的属性。 
{
    position: relative;
    top: -20px;
    left: 20px;
    background-color: white;
    width: 500px;
}
absolute的属性值必须要和relative一起使用，是相对于其父元素不动。 
{
    position: absolute;
    top: 120px;
    right: 0;S
    white: 300px;
    height: 200px;
}
