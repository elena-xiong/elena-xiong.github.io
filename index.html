如何用HTML做表格？(可添加内容的)

28/02/2018

1.开始要写<form action="http://...." method="POST"></form> action的属性值根据你要把表格发送的后台而改变。method 的属性值是固定不变的。

2.表格是由很多的快性元素组成的，如：div/p/h/ul/li... 

<div>
<label for="mingzi">Prenom</label>
<input type="text" name="mingzi" placeholder="名字">
</div>

注：
1,在input里面，可以加上placeholder="123@qq.com"的属性和属性值，其属性值 自己改变，来客人用户表格里该填写什么形式的内容。 2,切记for和name里面的属性值必须一样，type是规定了表格里面填写的是哪种形式的内容。
type有很多固定的属性值，其中最常用的是：
text:为了放一个普通的文本；
email:为了输入一个邮箱地址；
checkbox:勾选框；
file:为了上传文件；
radio:单选按钮，必须同时放几个才有用，这几个的name值必须一样；
submit:提交表格按钮，如果不需要默认的按钮内容，可以加上 value="..."这个属性。

如何用CSS把元素放中间？

25/02/2018

可以改变元素的margin-left和 margin-right,属性值用auto 会自动调整两边的边距。

注：auto这个属性值只能用于有固定的width和元素的display属性值是block的元素。

auto是调整这个大的元素，不会改变这个元素里面的文字，图片等等的位置。如果文字， 图片也需要改变，那么就需要对这些文字，图片等等做出相应的调整。 

<div>
    <p>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Suspendisse congue auctor quam, non fermentum mauris convallis 
        get. Nam at facilisis felis, quis cursus neque. Donec ut 
        suscipit massa. Morbi ultricies quam scelerisque, dignissim 
        velit eu, blandit nulla.
    </p> 
    <img src="fengjing.jpg"/>
</div>
                
div {
    width: 800px;
    margin-left: auto;
    margin-right: auto;
}
img {
    display: block;
}

如何用CSS把图片和文字放在同一行？

24/02/2018

需要先把图片和文字放在同一个块元素里，因为图片display 的默认值 是inline,本身就是按1,2,3这样排序的。但是文字p/h/ul/li...来标 写的，这些元素的display的默认值block,所以需要把 p/h/ul/li...这些元素的display 的属性值改成inline 就可以让图片和文字呈现横排排列了。 

<div>
    <img src="fengjing.jpg"/>
    <p>
        Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
        Suspendisse congue auctor quam, non fermentum mauris convallis 
        get. Nam at facilisis felis, quis cursus neque. Donec ut 
        suscipit massa. Morbi ultricies quam scelerisque, dignissim 
        velit eu, blandit nulla. 
    </p>
</div>
                
p {
    display: inline;
}

img {
    float: left;*/元素往哪里飘
}

如何用CSS把文字放在图片里面？

23/02/2018

1.需要先把图片和文字放在同一个块元素里，然后给块元素一个position: relative; 的属性和属性值。

2.给在文字所在的块元素一个position: absolute;的属性和属性值。

3.改变文字的display: block，再给文字一个top/bottom/left/right: 0/..px; 的属性和属性值，这样文字就会出现在图片里你设定的位置了。

注：只有块元素才会叠加

附加：如何有鼠标放在图片上有文字出来，反之看不到文字的效果？
1.需要给文字一个display: none;的属性和属性值，这样当鼠标不在 图片上时，文字是隐藏的。
2.需要给图片一个:hover {display: block;}的伪类属性，其属性值是改变:hover 后面的元素，这样鼠标放上去文字就会显现出来了。

    <div>
        <img src="fengjing.jpg">
        <span>我是一个小笨蛋，学习编程序好困难</span>
    </div>
                
    div {
        position: relative;
    }
    
    span {
        position: absolute;
        top: 0;
        display: none;
        background-color: rgb(241, 201, 142, 0.5);
    }
    
    img:hover + span  {
        display: block;
    }
